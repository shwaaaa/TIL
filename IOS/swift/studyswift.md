**이름짓기규칙**
------
함수나 메서드,상수 같은 것들은 소문자로 시작하는 camel case 사용
>- Lower Camel Case : fuction, method, variable, constant
>ex)someVariableName

>- class나 구조체, 열거형, extentionrkxdms 같은 타입 이름은 대문자로 시작하는 camel case 사용
>ex)Person,Point,Week

**변수와 상수**
----

상수 선언 키워드 **let**
변수 선언 키워드 **var**

상수 선언
let 이름: 타입 = 값

변수 선언
var 이름: 타입 = 값

값의 타입이 명확하다면 타입은 생략 가능
let 이름 = 값
var 이름 = 값
