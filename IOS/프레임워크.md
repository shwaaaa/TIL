**Foudation/UIKit**
---
**왜 import Foundation 또는 import UIKit이 필요할까?**
- Cocoa Touch Framework : 앱 개발 기초/필수적인 기능 등
- Media Framework : 다양한 미디어 기능 구현 필요 기능 등
- Core Services Framework : 시스템 서비스 제공 데이터 접근 등
- Core OS Framework : 보안/하드웨어 연동 등
